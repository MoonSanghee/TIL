고양이와 개는 몇 마리 있을까
```sql
SELECT ANIMAL_TYPE, COUNT(*)
FROM ANIMAL_INS
GROUP BY ANIMAL_TYPE
```

동명 동물 수 찾기
```sql
SELECT NAME, COUNT(*)
FROM ANIMAL_INS
WHERE NAME IS NOT NULL
GROUP BY NAME
HAVING COUNT(*)>1
ORDER BY NAME
```

입양 시각 구하기(1)
```sql
SELECT HOUR(DATETIME) HOUR, COUNT(*)
FROM ANIMAL_OUTS
WHERE HOUR(DATETIME)>8 and HOUR(DATETIME)<20
GROUP BY HOUR(DATETIME)
ORDER BY HOUR(DATETIME)
```
